開始前請先準備好
你的品牌色（主色、輔色、背景色的色碼）
開放的拍攝時段與每天最長可預約時數
要接收預約通知的 Email 信箱
Google 帳號（建立試算表與 Apps Script）

1
製作預約頁面
複製這段指令，貼到 Claude 對話框
請幫我製作一個線上預約系統 HTML 頁面，品牌色使用主色 （你的主色色碼）、輔色 （你的輔色色碼）、背景色 （你的背景色色碼）。功能需求如下： ・月曆選擇拍攝日期 ・選擇連續時段，最多 （X） 個時段（共 （X） 小時），時段為 （例：09:00–11:00、11:00–13:00、14:00–16:00） ・填寫姓名和聯絡電話 ・時段選擇區加上提醒文字：請確認後再預約，勿隨意更改 ・預約資料串接 Google 試算表（欄位：姓名、電話、預約日期、時段） ・已預約時段自動鎖住不能重複預約 ・有新預約時自動寄通知信到 （你的信箱）
2
設定 Google Apps Script
複製這段指令，繼續在 Claude 對話
請提供完整的 Google Apps Script 程式碼，功能包含： ・doGet：讀取試算表已預約資料，回傳 JSONP 格式，日期格式轉換為 yyyy-MM-dd，時區為 Asia/Taipei ・doPost：將預約資料寫入試算表（姓名、電話、日期、時段），並自動寄通知信到 （你的信箱）
①
建立 Google 試算表
新增空白試算表，第一列填入欄位標題：姓名電話預約日期時段，並將 C 欄格式設為「純文字」
②
開啟 Apps Script
試算表上方點「擴充功能」→「Apps Script」，把 Claude 給的程式碼全部貼上並儲存
③
部署為網頁應用程式
點「部署」→「新增部署」，類型選「網頁應用程式」，執行身分「我」，存取權「所有人」，點「部署」並複製產生的網址
3
上架到 GitHub Pages
複製這段指令，繼續在 Claude 對話
我已經有一個 HTML 預約頁面，請帶我一步步把它上架到 GitHub Pages，讓客戶可以透過公開網址使用。
①
註冊 GitHub 帳號
前往 github.com 免費註冊，完全不需要付費
②
建立 Repository 並上傳檔案
新增 Public Repository，上傳 HTML 檔案並命名為 index.html
③
開啟 GitHub Pages
Settings → Pages → Branch 選「main」→ Save，等約 1 分鐘即可取得公開網址
!
試算表 C 欄務必設為「純文字」格式，否則日期讀取會出錯，導致已預約時段無法正確鎖住
!
每次更新 Apps Script 程式碼後，需重新部署「新版本」，並將新網址更新到 HTML 檔案中
!
取消某筆預約時，直接在試算表刪除該列資料，客戶下次打開頁面時段就會自動重新開放
